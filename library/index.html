🔑 圖書館登入

首次使用請輸入 GitHub Personal Access Token(只存裝置、不上傳)。

如何取得 Token?
  1. 到 Fine-grained tokens 頁面
  2. 點「Generate new token」
  3. Repository access → Only select repositories → 選 jung_trilogy 與 taiwan-economic-history(兩個都要)
  4. Permissions → Repository permissions → Contents: Read-only
  5. 複製產生的 token(只會顯示一次)貼到下面
